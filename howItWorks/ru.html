Правила и условия

1. Общие положения

Настоящие правила регулируют отношения между сервисом Bazonix (далее - "Сервис") и пользователями, использующими услуги обмена криптовалют (далее - "Пользователи"). Используя наш сервис, вы автоматически соглашаетесь с данными правилами.

2. Порядок обмена

2.1. Сервис предоставляет услуги по обмену криптовалют в автоматическом режиме.
2.2. Курс обмена фиксируется в момент создания заявки.
2.3. Время обработки заявки зависит от загруженности сети выбранной криптовалюты.

3. Ответственность сторон

3.1. Сервис гарантирует выполнение обмена по зафиксированному курсу при условии корректного выполнения пользователем всех инструкций.
3.2. Пользователь несет полную ответственность за корректность указанных адресов и выбор сети.

4. Конфиденциальность

4.1. Сервис не хранит личные данные пользователей.
4.2. История обменов хранится в зашифрованном виде и доступна только по ID транзакции.
